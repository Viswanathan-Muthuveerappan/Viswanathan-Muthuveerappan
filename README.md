## 🚁 1. Metric Monocular Depth Estimation for Aerial Semantic Scene Completion

**Overview:** Robust aerial autonomy requires a comprehensive understanding of 3D environments. This project focuses on adapting and fine-tuning state-of-the-art vision foundation models to predict highly accurate, metric depth maps from single (monocular) aerial images. These depth predictions serve as a crucial geometric prior for downstream 3D Semantic Scene Completion (SSC), bridging the gap between 2D imagery and volumetric scene understanding. Notably, this work was conducted as part of the aerial SSC benchmark **OccuFly (CVPR 2026 Oral)**.

**Key Highlights:**
* Adapted vision foundation models (including Depth Anything V2 and Metric3D v2) to capture the complex, varying scale of aerial imagery across multiple altitudes, diverse environments, and seasonal changes present in the OccuFly dataset.
* Eliminated the need for heavy, expensive active sensors (like LiDAR) by extracting reliable geometric data directly from RGB images.

## 🗺️ 2. SegFly: A 2D-3D-2D Paradigm for Aerial RGB-Thermal Semantic Segmentation at Scale

**Overview:** A major bottleneck in aerial semantic segmentation is the lack of large-scale, high-fidelity annotated datasets. SegFly is an automated data generation framework designed to solve this data scarcity. By leveraging a novel 2D-3D-2D paradigm, this project autonomously generates dense, pixel-perfect semantic annotations for both RGB and Thermal aerial imagery at scale, requiring minimal manual effort.

**Key Highlights:**
* Implemented a depth-aware 3D-2D rendering pipeline to ensure high-fidelity semantic label projection.
* Synthesized large-scale RGB and aligned RGB-Thermal datasets from a small subset of manual annotations (under 3% manual supervision).
* Demonstrated that models trained on these generated pseudo-labels achieve robust out-of-distribution generalization.
